Http - interceptor

Stwórz interceptor, który będzie realizował cache postów z jsonPlaceholder.

Do pobrania danych użyj metody getPost(id: number) serwisu JsonPlaceholderApiService

Bonus: Interceptor powinien przechowywać w cache posty, które zostały pobrane w ciągu ostatnich 30 sekund.

Wynik

Pobrany post: {{ post | async | json }}
